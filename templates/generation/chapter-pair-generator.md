# Chapter Pair Generator Prompt (2-chapter beat)

Goal: generate CHxxxx (odd) + CHxxxx+1 (even) as a complete beat.

## Engine constraints

- Pleasure engine: B-type face-slap (mispricing -> re-pricing)
- Odd chapter = setup (mispricing action + pressure + hard cut)
- Even chapter = payoff (one decisive hammer + spectator re-pricing + bigger pressure hook)
- Minimal: pressure + visible gain + hook in EACH chapter

## Inputs

Paste the filled form from `deconstruct/two-chapter-form.md` here.

1) Chapter outlines:
- chapter-outline/CHxxxx.md
- chapter-outline/CHxxxx+1.md

2) Draft chapters:
- chapters/CHxxxx.md
- chapters/CHxxxx+1.md

3) Review checklists:
- reviews/CHxxxx.review.md
- reviews/CHxxxx+1.review.md

4) Update snippet for progress-tracking.md
